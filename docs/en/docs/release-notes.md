## Ravyn 3.x Series

### Ravyn 3.0.0 - The Rebrand

**Release Date:** TBD

Ravyn 3.0 marks the rebrand from Esmerald to Ravyn with a fresh new identity.

#### What's New

✨ **New Brand Identity**
- Renamed from Esmerald to Ravyn
- Updated package name and imports
- Fresh documentation and branding

✨ **Same Great Features**
- All Esmerald features preserved
- Seamless migration path
- 99% code compatibility

#### Migration Guide

See the [Migration Guide](./migration.md) for step-by-step instructions on upgrading from Esmerald to Ravyn.

#### Breaking Changes

- Package name changed from `esmerald` to `ravyn`
- Class names updated (`Esmerald` → `Ravyn`, `EsmeraldSettings` → `RavynSettings`)
- CLI commands changed from `esmerald` to `ravyn`
