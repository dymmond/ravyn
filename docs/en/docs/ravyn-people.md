## Contributors

We're grateful to everyone who has contributed to Ravyn through code, documentation, bug reports, and feature suggestions.

### How to Become a Contributor

Contributing to Ravyn is easy and welcoming:

1. **Report Bugs** - Found an issue? [Open a discussion](https://github.com/dymmond/ravyn/discussions)
2. **Suggest Features** - Have an idea? Share it in [discussions](https://github.com/dymmond/ravyn/discussions)
3. **Improve Documentation** - Fix typos, add examples, write guides
4. **Submit Code** - Fix bugs, add features, improve performance
5. **Help Others** - Answer questions, review PRs, share knowledge

See our [Contributing Guide](./contributing.md) to get started.
